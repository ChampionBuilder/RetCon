# RetCon

RetCon is an unofficial browser-based character planner for Champions Online.

It lets players build freeform or archetype characters, browse powers, assign advantages, manage travel powers and devices, and share builds through links or local saves.

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Project history

RetCon follows a long line of Champions Online planners, especially PowerHouse, HeroCreator, and the older championBuilder. The in-app About panel keeps the historical credits and links that helped preserve that lineage.
